## Files
- `index.html` — Home page

- `artifact1.html` — Artifact 1: Evolution of AI timeline
- `css/styles.css` — All styling
- `js/main.js` — Mobile nav + scroll animations

## Deploy to GitHub Pages (free, ~5 minutes)

1. Go to [github.com](https://github.com) and log in (create a free account if needed).
2. Click the **+** icon (top right) → **New repository**.
3. Name it something like `ai-ml-portfolio`. Set it to **Public**. Click **Create repository**.
4. On the new repo page, click **uploading an existing file**.
5. Drag in all the files from this folder (keep the `css` and `js` folders — GitHub will preserve the structure).
6. Click **Commit changes**.
7. Go to the repo's **Settings** tab → **Pages** (left sidebar).
8. Under "Build and deployment" → **Source**, select **Deploy from a branch**.
9. Under **Branch**, select `main` and folder `/ (root)`. Click **Save**.
10. Wait ~1 minute, then refresh the page — GitHub will show your live URL, something like:
    `https://yourusername.github.io/ai-ml-portfolio/`

That URL is what you submit for the assignment.

## Adding future artifacts

For each new artifact, duplicate `artifact1.html`, rename it (e.g. `artifact2.html`), replace the
content sections, and add a link to it in the `<ul class="nav-links">` block on every page (and
update the "Coming soon" card on the home page).
